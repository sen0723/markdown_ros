## 一、学习过程  
### 1.1 ros2入门之节点  
**1.1.1节点本质：**  
节点是操作系统里独立进程、可单独运行的可执行文件；支持跨语言开发（Python/C++ 等），不同语言节点可互相通信。  
**1.1.2ROS2 节点标准生命周期**  
1. 初始化 ROS2 接口（rclpy.init）  
2. 创建并初始化节点对象
循环执行节点业务逻辑（处理任务、日志输出）
3. 销毁节点、关闭 ROS2 接口（rclpy.shutdown）
4. 退出方式：Ctrl+C终止循环释放资源  
**1.1.3实操**  
   1. 面向过程写法  
`ros2 run learning_node node_helloworld`  
   2. 面向对象写法  
package.xml、setup.py配置，把 Python 脚本注册成可执行程序，colcon build编译操作。  
### 1.2Python入门  
**1.2.1、基础变量与数据类型**
1. **变量**：无需声明类型，直接赋值 `a = 10`；支持同时赋值 `a,b = 1,2`
2. **常用类型**
- 整数int、浮点数float、字符串str（单/双引号）、布尔bool（True/False）
- 列表list `[]`：可变有序；元组tuple `()`：不可变；字典dict `{k:v}`：键值对；集合set `{}`：去重

**1.2.2、运算符**
1. 算术：`+ - * / //取整 %取余 **幂`
2. 比较：`> < >= <= == !=`，结果为布尔值
3. 逻辑：`and or not`
4. 复合赋值：`+= -= *=`

**1.2.3、输入输出**
- 输出：`print(内容)`，多变量逗号分隔自动空格
- 输入：`input()` 读取字符串；转数字 `int(input())`  

**1.2.4分支判断 if**
```python
if 条件:
    语句
elif 条件:
    语句
else:
    语句  
```
## 二、遇到的困难及解决方法  
### 2.1GitHub SSH Key 配置  
**1. apt更新cdrom源报错**
  - 问题：虚拟机未挂载系统镜像，光盘源失效
  - 解决：删除系统内cdrom相关软件源配置  

**2. ROS密钥下载超时失败**
  - 问题：`GitHub raw`链接国内访问超时
  - 解决：清除旧镜像源，使用国内镜像密钥  

**3. ROS安装时虚拟机黑屏卡死** 
  - 问题：3D图形加速兼容冲突、安装占用资源过高
  - 解决：虚拟机设置关闭3D加速；安装损坏执行 `apt --fix-broken install` 修复   

**4. git push单独使用报错**
  - 问题：本地分支 `master` 与远程 `main` 名称不一致
  - 解决：固定推送指令  `git push origin master:main`  

### 2.2. VSCode Markdown预览渲染异常
  - 问题：工作区不受信任，插件被禁用；
  - 解决：信任当前工作区;
